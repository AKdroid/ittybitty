Welcome to ittybitty's documentation!
=====================================

ittybitty is a simeple python web framework. What makes itty bitty different
from exisiting python web frameworks is that ittybitty was not written with
the primary intent of serverving web pages.

ittybitty was developed to make it easy to add restfull api's to existing
application.


This documentation is still under development
