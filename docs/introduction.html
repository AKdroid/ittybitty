introduction: What is ittybitty¶

ittybitty is yet another python web framework. However it aims to be different from other web in that it primary aim is not to serve up html pages to a web brwser. (Although it can also do this.) Ittybitty’s main purpose is to allow you to add public api’s to an exisiting python apilication with minimal effort.

To serve this puport it has a number of usefull features ...
  • Automatic API reference and documentation
  • Built in suport for multiple authentication mechanisms
  • Built in support for json/xml data transfer.
